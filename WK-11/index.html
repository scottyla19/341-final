Super"HERO" Images

Hero images are typically used as full screen backgrounds that take up the entire width and height of the screen and are a very common current web design practice. I decided to combine hero images with another current popular web design concept of parallax, or scrolling, images.

Mutliple Parallax

This page is created with three images each within a parallax wrapper that is set to absolut. Each parallax group is set to relative and take up 100vh. Finally within each group is a specific parallax layer with various z transforms applied. The deep is -2px, base is -1px, and base is 0px in the z transform.
