What is Responsive Web Design?

Responsive web design (RWD) is the design and development process of being aware of the user experience of your site on various sized screens. By focusing on the smallest screen (mobile) we can create websites that work well on the most difficult to design for and popular mobile screen sizes. Using media queries we can modify how the website appears on varying screen sizes.

Why Responsive Web Design?

RWD makes sense for both the developers and the users. For the developers they can plan ahead and develop one website that is a great experience for users no matter what device the users are on. For the users they get a great experience on all of their devices.
